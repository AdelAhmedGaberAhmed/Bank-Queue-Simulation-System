# Bank-Queue-Simulation-System
C++ bank queue simulation system using custom Queue implementation with Date and String utility libraries to manage ticketing and customer flow.


# 🎟️ Queue Bills - Bank Queue Management System

## 📌 Overview
This project is a C++ console-based simulation of a bank queue system. It manages customer entry using a Queue data structure and simulates real-life banking waiting systems such as ticket generation, queue tracking, and service order management.

The system also includes custom utility libraries like **Date** and **String**, implemented from scratch.

---

## 🛠️ Technologies Used
- C++
- Object-Oriented Programming (OOP)
- Custom Data Structures (Queue)
- Dynamic Memory Management
- Console Application

---

## 🧱 Custom Libraries

### 📅 Date Library
- Handles date operations
- Used for tracking time-based events in the system

### 🔤 String Library
- Custom implementation of string operations
- Used instead of built-in string handling

---

## 📊 Core Data Structure

### 📥 Queue System
- FIFO (First In First Out) logic
- Manages customer tickets
- Tracks:
  - Current serving number
  - Remaining customers in queue
  - Total served customers

---

## ⚙️ System Features
- Generate customer tickets automatically
- Display current queue status
- Show number of people waiting
- Track how many customers have been served
- Simulate real bank queue behavior
- Continuous system monitoring in console
